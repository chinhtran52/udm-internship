## letter for file name:
### Type of walking
    1. Normal   : n
    2. Fixed    : f
    3. Half     : h
### Type of curvature:
    1. Gaussian : g
    2. Mean     : m
    3. Absolute : a
    4. RMSquare : r
### Mesh format:
    PLY : Polygon File format
### Convention:
```
code = <type of walking> + <type of curvature> + <pose number> + <.ply>
```
### Example:
- ng1.ply = normal walking Gaussian curvature pose 1 mesh format
